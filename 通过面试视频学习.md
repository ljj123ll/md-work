## 举例
如何渲染1000w行表格？

1. 背景：数据足够大，大数据公司，需要在前端展示大量数据，以防住卡顿，所以我们想要通过更好的方案来实现这需求。
2. 我作为负责人，调研这部分方案
	1. dom 
	2. 虚拟表格
	3. canvas table
	4. 可视区绘制算法优化
	5. canvas 结合 Webassembly 技术实现（skia + Webassembly）

3. 方案落地：我封装了画布表格引擎，解决了上述问题，并且实现了1000w行数据加载流畅交互
4. 在这个过程中遇见了什么问题，我是怎么解决上述问题的。
